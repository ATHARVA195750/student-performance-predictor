# Student Performance Predictor

Predicts whether a student will pass or fail based on academic 
and demographic data.

## Results
- Accuracy: 91.1%
- Algorithm: Random Forest (100 estimators)
- Dataset: 395 students · 33 features (UCI Student Performance)

## Key Findings
- G2 (mid-year grade) is the strongest predictor of final outcome
- Absences and past failures are the next most important factors
- Family education level (Medu, Fedu) also plays a measurable role

## Tech Stack
Python · scikit-learn · Pandas · NumPy · Matplotlib

## Run it yourself
Open the .ipynb in Google Colab and run all cells.
Dataset loads automatically from UCI ML Repository.
